🌤️ Weatherly Overview

Weatherly is a weather app that provides real-time weather conditions and 7-day forecasts for cities around the world.

Key Features

  • 🌡️ Current Weather
  • 📊 Weather Metrics (Humidity, Wind, Visibility, Rain)
  • 📅 7-Day Forecast
  • 📈 Temperature Trends
  • 🗺️ Interactive City Map

APIs and Libraries

  • Open-Meteo (Weather Forecast and Geocoding APIs)
  • Tomorrow.io Weather Icons
  • OpenStreetMap
  • Chart.js
  • Leaflet.js
